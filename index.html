Install the Watasu CLI

Watasu CLI.
Work from your terminal.

Use watasu to log in, manage apps, and work with Watasu from the command line. Install it below, then sign in and get started with your account on watasu.io.

macOS / Linux / WSL

curl -fsSL https://watasuio.github.io/watasu-cli/install.sh | bash

Windows PowerShell

irm https://watasuio.github.io/watasu-cli/install.ps1 | iex

Homebrew

brew tap watasuio/watasu
brew install watasu
